# Template Eyebrow Labels

The hook slide on every carousel displays a small amber eyebrow label above the drug name. This label identifies the template and is auto-generated by the design skill — the script never writes it.

| Template | Eyebrow Label |
|----------|--------------|
| 1 | Drug Breakdown |
| 2 | When Not to Give |
| 3 | How It Works |
| 4 | After You Push It |
| 5 | Why You Give It |

## Rendering

The eyebrow maps to `.hook-eyebrow` in the HTML. It renders in amber (`#daa040`), uppercase, letter-spaced, at ~4.4cqw. See `design_mockup.html` for exact CSS.
